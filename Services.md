### Basic Cloud Operations Package
This package gets a company started with PCs, email, and colaboration tools with an easy per user, per month subscription.

### Basic On-prem Operations Package
This package gets a company started with PCs, email, and colaboration tools also, adding on-prem assets like domain and file servers.

### Email Security Package
This package hardens an organizations email security to prevent fraudulant emails, reduce SPAM and Phishing emails, and automate training campaigns for users.

### Global Network Package
This package includes configuration, security, and administration of an organization's global network, distributed on-prem sub-nets, vpn secure access, and protection of production assets.

### Enterprise Systems Package
This package includes configuration, security, and administration of an organization's ERP, MES, IoT, CAD, PDM, CRM, and similar systems.

### Phones and Printers Package
This package adds routable phone numbers with an option for phones and printers to an organization's technology assets.
